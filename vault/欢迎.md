# ForHermes Vault

AI-native 个人知识库。由 clawliu (人类) 和 Hermes Agent (AI) 共同维护。

## 快速导航

| 目录 | 用途 | 负责人 |
|------|------|--------|
| [[diary/]] | 每日互动记录（事务流水账） | AI |
| [[inbox/]] | 快速捕获，待整理 | 任意 |

| 目录 | 用途 | 负责人 |

## 工具集成

- **训记 API** — `xunji-trains` / `xunji-diet` / `xunji-body` skills
- **Obsidian** — 通过 `obsidian` skill 读写
- **Apple Notes/Reminders** — 通过 `apple-notes` / `apple-reminders` skills
- **QQ Bot** — 随时通过 QQ 对话

## 维护约定

> [!important] 分工原则
> - `memory/`：系统写，只看不改
> - `knowledge/`：clawliu 指定才写
> - `my-knowledge/`：AI 自主判断写入（Claim-Evidence 格式）
> - `diary/`：每次会话写一条
> - `inbox/`：快速捕获，定期分流
> - `archive/`：项目完成后移入并加索引

> [!tip] 写作规范
> - 所有笔记建议加 frontmatter（tags + date/created + status）
> - 新建使用 [[templates/|对应模板]]
> - 跨目录引用用 wikilink `[[路径/文件名]]`
